# Anvil tutorial from Blender Guru

Video [Link](https://www.youtube.com/watch?v=yi87Dap_WOc)

## Blender tools used

| Name | shortcut |
| - | - |
| Move | G |
| Move only in one direction | G + [XYZ] |
| Scale | S |
| Scale exclude axis | S + Shift + [XYZ] |
| Extrude | E |
| Loop cut | Ctrl + R |
| Proportional Editing | O |
| Hide | H |
| Un-hide | Alt + H |
