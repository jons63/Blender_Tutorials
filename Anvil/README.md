## Blender tools used part 1

| Name | Shortcut |
| - | - |
| Move | G |
| Move only in one direction | G + [XYZ] |
| Scale | S |
| Scale exclude axis | S + Shift + [XYZ] |
| Extrude | E |
| Loop cut | Ctrl + R |
| Proportional Editing | O |
| Hide | H |
| Un-hide | Alt + H |
